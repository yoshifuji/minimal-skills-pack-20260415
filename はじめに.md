# はじめに — Tutor Marketplace Example

このファイルは、legacy tutor marketplace overlay をたどるための入口です。新しい service marketplace を始める場合は `README.md` から `marketplace-foundation-workflow` を使ってください。

## まずやること

1. `skills/example-tutor-marketplace/SKILL.md` を読む
2. `docs/講師マーケットプレイス実行手順書.md` を読む
3. `references/examples/tutor-marketplace/データモデル概要.md` を読む
4. まだコードは書かず、`SPEC.md` と `tasks/` を先に作る

## 実装順

1. schema / triggers / RLS
2. signup / login / logout / middleware
3. public pages / tutor dashboard / student dashboard
4. Stripe Connect onboarding
5. booking + Checkout
6. webhook reconciliation
7. refund / cancellation
8. security hardening
9. deploy / launch

## 迷ったら

- 実装順は `skills/example-tutor-marketplace/SKILL.md` を優先
- DB は `references/examples/tutor-marketplace/データモデル概要.md`
- Stripe は `references/Stripeイベントマトリクス.md`
- env は `references/環境変数とシークレット一覧.md`
- 最終確認は `references/examples/tutor-marketplace/実行成果物チェックリスト.md`
