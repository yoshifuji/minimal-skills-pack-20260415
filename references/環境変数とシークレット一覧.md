# 環境変数とシークレットのマトリクス

| 変数 | スコープ | 例 | 用途 |
|---|---|---|---|
| `NEXT_PUBLIC_APP_URL` | public + server | `http://localhost:3000` | success / cancel / return URL |
| `NEXT_PUBLIC_SUPABASE_URL` | public + server | `https://xxx.supabase.co` | SSR / browser client |
| `NEXT_PUBLIC_SUPABASE_PUBLISHABLE_KEY` | public + server | publishable key | browser + SSR auth |
| `SUPABASE_SECRET_KEY` | server only | secret key | admin write、webhook、join |
| `STRIPE_SECRET_KEY` | server only | `sk_test_...` | Stripe server SDK |
| `STRIPE_WEBHOOK_SECRET` | server only | `whsec_...` | webhook 署名検証 |
| `STRIPE_CONNECTED_ACCOUNT_COUNTRY` | server only | `JP` | Connect account 作成 |
| `NEXT_PUBLIC_PLATFORM_CURRENCY` | public + server | `jpy` | 表示通貨と line item 通貨 |
| `PLATFORM_FEE_BPS` | server only | `1500` | application fee 計算 |

## デプロイ時の注意

- 初回デプロイ時は `STRIPE_WEBHOOK_SECRET` なしでも画面は上がる
- 本番 URL が決まったら Supabase の `Site URL` / `Redirect URLs` を更新する
- Stripe webhook endpoint 作成後に `STRIPE_WEBHOOK_SECRET` を登録し、再デプロイする
